# Scala Coding Exercise - Search Application 

This code has been tested using Scala 2.12.3 and Java 8.

It's an SBT project

## Tests

Tests can be run using the following shell command:

```
sbt test
```

## Usage

Application can be run by using directly from shell the following command:

```
sbt "run path-of-directory-where-text-files-are "
```

### Example 

```
sbt "run examples"
```


NOTE: to exit search console type <b>:quit</b>
